## Installation

### Prebuilt binaries

Watch the [releases](https://github.com/YXL76/VideoMosaic/releases) page.

### Build from source

#### Requirements

- [Rust](https://www.rust-lang.org/) nightly
- [FFmpeg](https://www.ffmpeg.org/)

#### Build

```shell
git clone https://github.com/YXL76/VideoMosaic.git
cd VideoMosaic
cargo build --release
```
